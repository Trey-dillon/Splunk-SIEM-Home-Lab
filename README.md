# 📊 Splunk SIEM Home Lab

## Project Overview

This project documents the setup and use of Splunk as a Security Information and Event Management (SIEM) platform.

The lab focuses on collecting, searching, and analyzing Windows security events to simulate basic Security Operations Center (SOC) workflows.

### Skills Demonstrated

- SIEM fundamentals
- Windows log analysis
- Splunk Search Processing Language (SPL)
- Authentication monitoring
- Event ID analysis
- Failed login investigation
- Basic security detection logic

---

# Lab Environment

- Splunk
- Windows 11
- Windows Security Event Logs
- Virtualized home lab environment
